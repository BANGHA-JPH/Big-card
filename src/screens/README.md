# Screens

This directory contains all application screens organized by feature.

## Structure

- **auth/** - Authentication screens (Login, Register, ForgotPassword)
- **home/** - Home screen and featured products
- **shop/** - Shopping screens (Search, Category, Product detail)
- **cart/** - Cart and checkout screens
- **orders/** - Order listing and detail screens
- **account/** - User profile and account management

## Adding a New Screen

1. Create a folder with the screen name
2. Create the screen component file
3. Add navigation integration
4. Import and use in navigation configuration
